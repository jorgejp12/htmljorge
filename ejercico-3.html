Los tres pilares de la Web

La Web fue creada por Tim Berners-Lee y se basa en tres tecnologías fundamentales:

  • HTML
  • HTTP
  • URL

HTML

El lenguaje de marcado que permite la estructuración del contenido en la web.

Fuente: HTML

HTTP

Protocolo de comunicación que permite la transferencia de información en la Web.

Fuente: HTTP

URL

Identificador único para acceder a recursos en la Web.

Fuente: URL
